# Swedish Space Ecosystem — Interactive Visualizations

Interactive knowledge graph visualizations mapping Sweden's space ecosystem: research institutions, companies, international collaborations, and funding networks.

**290 nodes · 474 edges · 9 data sources**

## Visualizations

### [3D Ecosystem Explorer](index.html) — 8 Interactive Views

| View | Content | Nodes |
|------|---------|-------|
| 1. Thematic Graph | 40 research themes across 8 domains | ~40 |
| 2. Institutions | 23 Swedish + 45 international institutions | ~68 |
| 3. Researchers | 150 researchers, co-authorship network | ~150 |
| 4. System Graph | 1,716 researchers + affiliations (radial) | 1,716 |
| 5. Industry Graph | 79 organizations + 4 funders | ~83 |
| 6. Knowledge Graph | 187 nodes · 482 edges · 5 sources | 187 |
| 7. International | 40 institutions · 35 SE orgs · Horizon Europe | ~115 |
| 8. Collaboration | 83 actors · 284 collaboration links | 83 |

### [Combined Ecosystem Map](combined.html)

2D combined ecosystem map with pan/zoom.

## Technology

- **Three.js r128** — 3D rendering
- **D3.js v7** — Force simulation and 2D views
- Standalone HTML files — no server required, no install needed

## Context

Part of a Vinnova excellence cluster initiative analyzing Sweden's space sector. Core finding: **"Strong nodes. Weak links"** — Sweden has world-class research and infrastructure, but extreme market concentration (HHI 0.55 in core segment) and 71% of companies operating in isolation.
